# SQL Cheat Sheet (Quick Reference)

## Data Types
- Int
- Varchar
- Datetime
- String
- Char 
- Boolean
- JSON



## Operators
- **Logical operators**
  - AND
  - OR
  - NOT
  - ALL
  - ANY
  - BETWEEN
  - EXISTS
  - IN
  - LIKE
  - SOME
 
- **Arithmetic operators**
  - ADD
  - SUBSTRACT
  - MULTIPLY
  - DIVIDE
  - MODULE

- **Bitwise operators**
   - (&) Bitwise AND
   - (|) Bitwise OR
   - (^) Bitwise exclusive OR

- **Comparision operators**
   - (=) Equal to
   - (>) Greater than
   - (>=) Greater than or equal to
   - (<=) Less than or equal to
   - (<>) Not equal to 

## Data Manipulation Language (DML) Statements ##
  - INSERT
  - UPDATE
  - DELETE
  - MERGE
  - CALL
  - EXPLAIN PLAN
  - LOCK

## Data Definition Language (DDL) Statements ##

## Data Control Language (DCL) Statements ##

- REVOKE

## Transaction  Control Language (TCL) Statements ##  
- COMMIT
- SAVEPOINT
- ROLLBACK
- SET TRANSECTION
- SET CONSTRAINT
