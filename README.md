# Data Types
- Int
- Varchar
- Datetime
- String
- Char

- JSON


# Operators
- **Logical operators**
  - AND
  - OR
  - NOT
  - ALL
  - ANY
  - BETWEEN
  - EXISTS
  - IN
  - LIKE
  - SOME

    
- **Arithmetic operators**
  - ADD
  - SUBSTRACT
  - MULTIPLY
  - DIVIDE
  - MODULE

- **Bitwise operators**
   - (&) Bitwise AND
   - (|) Bitwise OR
   - (^) Bitwise exclusive OR


- **Comparision operators**
   - (=) Equal to
   - (>) Greater than
   - (>=) Greater than or equal to
   - (<=) Less than or equal to
   - (<>) Not equal to
